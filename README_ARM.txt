  Basler Vision System - ARM 安裝 (Rock 5B / Debian 12)
═══════════════════════════════════════════════════════════════════

快速安裝（3 步驟）：

1. 解壓縮專案並進入目錄
   git clone https://github.com/你的用戶名/Real-time_item_monitoring_system.git
   cd Real-time_item_monitoring_system

2. 運行安裝腳本
   chmod +x install_arm.sh
   ./install_arm.sh

3. 啟動應用
   ./run.sh

─────────────────────────────────────────────────────────────────

腳本會自動：
✓ 檢查系統架構
✓ 安裝 Qt 圖形庫
✓ 創建 Python 虛擬環境
✓ 安裝所有 Python 依賴
✓ 創建快速啟動腳本
